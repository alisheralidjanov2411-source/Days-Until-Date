# Days Until Date

This Python program calculates how many days are left until a specific date.

## Features
- User-defined target date
- Calculates remaining days
- Displays result in a simple format

## How it works
The user enters a date, and the program calculates the difference between the current date and the target date.

## Technologies
- Python (basic)
- Datetime module

## Purpose
This project was created to practice working with dates and time calculations.

## Future Improvements
- Add countdown timer
- Handle different date formats
- Create GUI version
